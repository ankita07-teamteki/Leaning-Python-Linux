# Project Folder Overview

This directory gives a quick map of all main folders in this project.

## Main Folders

- `linux` - Linux learning notes (`first_day.md`, `second_day.md`)
- `python` - Python learning notes (`first_day.md`, `second_day.md`, `third_day.md`)
- `apache` - Apache Docker setup and static website files
- `react-app` - React app source code for Docker container
- `project_overview` - This overview folder

## Main Root Files

- `docker-compose.yml` - Runs Apache and React containers together
- `DOCKER_RUN.md` - Docker setup and run instructions
